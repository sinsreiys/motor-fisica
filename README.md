# 🫧 Motor de Física 2D

Interactive 2D physics engine built from scratch in vanilla JavaScript — no libraries, no frameworks.

🔴 **[Live Demo](https://sinsreiys.github.io/motor-fisica)**

---

## Features

- Rigid body dynamics with mass and velocity
- Elastic collision detection and resolution (impulse-based)
- Fixed-timestep game loop (60 FPS)
- Click & drag to launch bubbles
- Random colored bubbles with radial gradient rendering
- Grid background with linear gradient

## Tech Stack

- Vanilla JavaScript (ES6)
- HTML5 Canvas API
- Zero dependencies

## How to run

Just open `index.html` in any browser — no build step needed.

## Physics concepts implemented

- Newton's second law (F = ma)
- Impulse-based collision response
- Overlap separation to prevent tunneling
- Coefficient of restitution (elasticity)
